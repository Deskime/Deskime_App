# Deskime Desktop TimeWidget

![Screenshot](./screenshot.png)

A TimeWidget for your Desktop made with Python.

------------

## Features

-  **Custom Desktop TimeWidget**: Located at the top-center, this strategic app helps make your desktop look cleaner, minimalist and helps in better time check.
  

### Tested On

- **Python Version**: Python 3.10 (64-bit)

### Prerequisites

- [x] Python 3.8 or higher
- [x] Pythonw (typically installed with Python, used for console-less execution)
- [x] pip
- [x] Windows 10 or higher (currently not supported on Mac)
